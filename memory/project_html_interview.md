所有总结成 HTML 的学习笔记，必须自动加上一节 **"面试怎么回答 / 面试话术"**。

**Why:** 用户整理 HTML 总结时，面试问答是刚需——如果我不加，用户要专门再提一次。

**How to apply:** 每次新建或追加 HTML 学习总结时，在内容之后、总结 section 之前（或作为压轴 section）插入面试话术。格式参照已有的第 8 节：用 `.code` 块展示可直接背诵的回答，语言体现"架构思维/按需升级/边际收益"这类面试官喜欢听的词。
