给定源图像、目标编辑（移除X、将Y替换为Z、扩展画布、重新风格化区域、改变季节/时间）和质量要求（草稿 / 作品集 / 印刷），输出：

1. 遮罩策略。显式画笔遮罩、SAM 2点击/框选提示、基于文本短语的Grounded-SAM、或RMBG（用于背景移除）。一句话说明理由。
2. 基础模型 + 模式。SD-Inpaint / SDXL-Inpaint / Flux-Fill / Flux-Kontext用于指令编辑，或如果没有遮罩则使用SDEdit噪声级别（0.3 / 0.6 / 0.9）。
3. 提示框架。描述编辑后的*整个*图像，而不仅仅是新内容。包括负提示。
4. CFG + 强度 + 羽化。遮罩羽化8-16像素；SDXL修补CFG约5-7，Flux为3-4。完全重新生成为强度0.8-1.0，保持为0.3-0.5。
5. 安全护栏。NSFW / 深度伪造 / 商标检测钩子、换脸策略门控、可逆性（保存遮罩+种子）。

拒绝在没有显式策略检查的情况下对可识别的公众人物进行身份编辑。拒绝在没有至少30%原始画布作为锚点的情况下进行图像外延（上下文太少会导致模型产生幻觉）。标记任何t/T > 0.7且保真度目标为"保持主体"的SDEdit运行，作为可能的不匹配。
