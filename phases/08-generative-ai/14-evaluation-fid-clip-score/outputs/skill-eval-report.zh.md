给定一个新的生成模型检查点、一个参考基线和一种模态（图像 / 视频 / 音频 / 3D），输出完整的评估计划：

1. 样本质量。在 10-30k 样本上对保留真实集的 FID / FD-DINO / CMMD。匹配分辨率。报告 3 个种子的均值 ± 标准差。
2. 遵循度。提示-图像对上的 CLIP 分数 / CMMD。对于文本到图像，包含 HPSv2 + ImageReward + PickScore。对于视频，添加视觉语言指标（V-Eval）。对于音频，使用 CLAP + MOS。
3. 成对偏好。对 200-2000 个提示与基线进行盲测 A/B。包含人类 + LLM 评估 + PartiPrompts 覆盖。
4. 类别细分。每个提示类别（人物、动物、文字渲染、构图、风格）的性能。即使全局指标改善，也要标记每个类别的退化。
5. 安全 / 滥用。NSFW 分类器、深度伪造检测器、水印检查、Top-K 生成上的版权相似度扫描。
6. 签字。明确的门控：FID 在基线的 +5% 以内 OR 超过 55% 的人类胜率 OR 有记录的定性优势。不得使用单一指标声明。

拒绝报告 N < 5000 的 FID。拒绝交付基于模型可能在训练中见过的提示计算的基准结果。拒绝报告仅有 LLM 评估结果而没有人工交叉验证的结果。标记任何声称某个指标"提升了 20%"但不报告绝对基值和单一种子的结果。
