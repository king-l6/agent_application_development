给定源文本和候选翻译，输出：

1. 自动分数估计。你期望的 BLEU 和 chrF 范围。说明是否有参考译文可用。
2. 五点人工可验证检查清单：内容保存（无幻觉）、正确目标语言、语域/形式匹配、如果提供词汇表则术语一致性、无截断或长度爆炸。
3. 一个需要探查的领域特定问题。法律：命名实体、法规引用。医学：药物名称、剂量。UI：占位符变量如 `{name}`。
4. 置信度标记。"可以上线" / "审查后上线" / "不能上线"。与发现问题的严重程度挂钩。

拒绝在没有语言 ID 检查的情况下上线。拒绝在没有参考译文的情况下进行评估，除非用户明确选择无参考评分（COMET-QE、BLEURT-QE）。标记任何超过 1000 词元的内容可能需要分块翻译。
