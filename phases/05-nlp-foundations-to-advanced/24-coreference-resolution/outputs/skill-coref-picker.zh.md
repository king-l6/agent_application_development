给定一个用例（单文档 / 多文档、领域、语言），输出：

1. 方法。基于规则 / 神经跨度方法 / LLM 提示 / 混合方法。一句话说明理由。
2. 模型。如果是神经方法，给出具体的模型 checkpoint。
3. 集成。操作顺序：分词 → NER → 共指 → 下游任务。
4. 评估。在留出集上计算 CoNLL F1（MUC + B³ + CEAF-φ4 平均值）+ 对 20 篇文档进行人工簇审核。

对于超过 2,000 个 token 的文档，拒绝使用仅 LLM 的共指方法（除非使用滑动窗口合并）。拒绝任何在共指时没有提及级别精确率-召回率报告的流水线。标记在人群多样性文本中部署的性别启发式系统。
