# SVM 核选择指南

SVM 由两个选择定义：核（决定决策边界的形状）和正则化参数（控制间隔宽度与分类错误之间的权衡）。正确设置这些是无用模型与强大模型之间的区别。

## 决策清单

1. 数据是线性可分的（或接近）吗？
   - 是：使用线性核。它更快且更具可解释性。
   - 否：进入第 2 步。

2. 特征与样本相比如何？
   - 特征 >> 样本（例如，带 TF-IDF 的文本）：使用线性核。高维数据通常是线性可分的。RBF 增加复杂度而无收益。
   - 样本 >> 特征（例如，10-50 个特征的表格数据）：RBF 核是默认选择。

3. 决策边界预期是平滑的吗？
   - 平滑、连续的边界：RBF 核
   - 多项式形状的边界：多项式核（从 2 或 3 次开始）
   - 领域知识提示特定的交互项：使用匹配次数的多项式核

4. 数据集有多大？
   - 少于 10,000 个样本：任何核都可以，RBF 是安全的默认选择
   - 10,000 到 100,000：线性核或 LinearSVC（原问题形式，每轮 O(n)）
   - 超过 100,000：不要使用核 SVM。切换到线性 SVM、梯度提升或神经网络。

5. 你缩放特征了吗？
   - SVM 需要特征缩放。始终在拟合前进行标准化（零均值、单位方差）。未缩放的特征会扭曲间隔的几何形状。

## 核选择流程图

```
开始
  |
  v
特征 > 1000 或特征 >> 样本？
  是 --> 线性核（为速度用 LinearSVC）
  否  --> 数据集 < 10k 样本？
            是 --> 先尝试 RBF（最佳通用核）
            否  --> 线性核（核 SVM 是 O(n^2) 到 O(n^3)）
```

如果 RBF 效果不好，尝试多项式 2-3 次。如果仍不行，问题可能不适合 SVM。

## 调整 C（正则化）

C 控制错误分类的惩罚。它与正则化强度成反比。

| C 值 | 效果 | 何时使用 |
|---------|--------|-------------|
| 0.001 - 0.01 | 宽间隔，允许大量违规 | 有噪声的数据，希望泛化 |
| 0.1 - 1.0 | 平衡 | 良好的起始范围 |
| 10 - 1000 | 窄间隔，很少违规 | 干净的数据，需要高准确率 |

调整策略：
- 从 C=1.0 开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10, 100, 1000]
- 使用交叉验证选择最佳值
- 如果最佳 C 在范围的边缘，向该方向扩展范围

## 调整 gamma（RBF 核）

Gamma 控制单个训练点的影响力范围。它定义了高斯核的宽度。

| gamma 值 | 效果 | 何时使用 |
|-------------|--------|-------------|
| 小 (0.001) | 每个点影响大面积。平滑、简单的边界 | 欠拟合或特征少 |
| 中 (auto: 1/n_features) | sklearn 默认值。合理的起始点 | 通用 |
| 大 (10+) | 每个点只影响附近点。复杂、波动的边界 | 有过拟合风险 |

调整策略：
- 从 gamma="scale"（1 / (n_features * X.var())，sklearn 默认值）开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10]
- 低 gamma + 高 C 倾向于过拟合
- 高 gamma + 低 C 倾向于欠拟合

## 联合调整 C 和 gamma

C 和 gamma 会相互影响。始终一起调整，而不是独立调整。

推荐方法：
1. 粗略网格搜索：C 在 [0.01, 0.1, 1, 10, 100]，gamma 在 [0.001, 0.01, 0.1, 1, 10]（25 种组合）
2. 找到最佳区域
3. 在最佳区域附近进行精细网格搜索（例如，C 在 [5, 10, 20, 50]，gamma 在 [0.05, 0.1, 0.2]）
4. 全程使用 5 折交叉验证

## 常见错误

- 在高维稀疏数据上使用 RBF 核（线性更好且快 100 倍）
- 忘记缩放特征（最常见的 SVM 错误）
- 在有噪声的数据上将 C 设置得太高（记住噪声而非学习边界）
- 在超过 50k 样本的数据集上使用核 SVM（训练时间过久）
- 不一起调整 C 和 gamma（它们相互补偿）
- 默认使用 5 次以上的多项式核（过拟合严重，先尝试 2 或 3 次）

## 快速参考

| 核 | 何时使用 | 关键参数 | 训练复杂度 |
|--------|------------|----------------|-------------------|
| 线性 | 文本/TF-IDF、特征多、数据量大 | 仅 C | 每轮 O(n) |
| RBF | 通用、少于 10k 样本 | C, gamma | O(n^2) 到 O(n^3) |
| 多项式 | 已知多项式关系 | C, degree, coef0 | O(n^2) 到 O(n^3) |
| Sigmoid | 很少有用（等价于两层神经网络） | C, gamma, coef0 | O(n^2) 到 O(n^3) |
