# 模型路由作为成本降低原语

> 一个动态代理评估每个请求（任务类型、token 长度、嵌入相似度、置信度），将简单查询发送到廉价模型，将复杂查询升级到前沿模型。也称为模型级联。生产案例研究显示，在 US/UK/EU 部署中，在保持同等质量的情况下成本降低20-60%；高流量 SaaS 上的30%路由效率改进可转化为六位数的年度节省。2026年的背景是 LLM 推理价格每年下降约10倍——从2022年底到2026年，GPT-4 级别的 token 从每百万20美元降至约0.40美元。大部分下降来自更好的服务栈（第17阶段·04-09），而非硬件。路由是你如何在不造成产品回归的情况下将价格下降转化为利润空间。失败模式是廉价模型漂移：路由将40%推到较弱的模型上，推理任务的质量下降3-5%，没有一个季度被注意到。通过在线质量指标（而非仅离线评估集）来把关路由。

**类型：** 学习
**语言：** Python（标准库，玩具级级联路由模拟器）
**前置知识：** 第17阶段·01（托管 LLM 平台），第17阶段·19（AI 网关）
**时间：** 约60分钟

## 学习目标

- 解释模型级联：优先使用廉价模型进行置信度检查，在低置信度时升级。
- 列举四种路由信号（任务分类、提示长度、与已知困难集的嵌入相似度、首轮自置信度）。
- 计算在目标路由拆分和质量损失容忍度下的预期混合成本。
- 指出捕捉廉价模型蔓延的漂移监控指标（在线质量门）。

## 问题

你的服务每月在 GPT-5 上花费8万美元。你的分析显示70%的查询很简单："巴黎现在几点？""改写这个句子。"Haiku 级别模型能以3%的成本完美处理这些查询。30%需要 GPT-5 的推理能力——编码、数学、多步骤规划。

如果你将70%路由到廉价模型，30%路由到昂贵模型，你的账单在产品质量不变的情况下下降约65%。这就是路由。关键在于构建代理时不影响质量。

## 概念

### 四种路由信号

1. **任务分类**：简单/复杂/代码生成/数学/聊天。可以是基于规则的分类器、小型 LLM（Haiku 级别，每百万0.25美元）或与标记桶的嵌入相似度。输出：路由 = 廉价 / 平衡 / 前沿。

2. **提示长度**：>4K tokens 的提示通常需要前沿模型来保持连贯性。<500 tokens 的提示通常不需要。

3. **与已知困难集的嵌入相似度**：如果查询与已知困难桶接近（余弦 > 0.88），直接升级到前沿模型。

4. **首轮自置信度**：先发送给廉价模型；如果模型的 log-prob 显示低置信度，或者它拒绝回答，或者输出模棱两可的语言，就在前沿模型上重试。这会使约10%流量的 P95 延迟增加，但为另外90%节省50%以上的成本。

### 三种模式

**预路由**（前置分类器）：增加约5-10毫秒延迟；整体速度最快。

**级联**（先用廉价模型，低置信度时升级）：中位延迟约1.2倍（廉价运行加验证），升级时约2倍。质量底线最佳。

**集成路由**（对样本并行运行廉价和前沿模型，让奖励模型选择）：质量最高，成本最高；仅用于关键的 A/B 测试。

### 实现

AI 网关（第17阶段·19）暴露路由。LiteLLM 有带回退和成本路由的 `router` 配置。Portkey 有防护 + 路由。Kong AI Gateway 有基于插件的路由。OpenRouter 的模型市场暴露推荐 API。

开源：RouteLLM（LMSYS）、Not Diamond（商业）、Prompt Mule。

### 2026年价格曲线

| 模型类别 | 2022年底 | 2026年 | 变化 |
|----------|----------|--------|------|
| GPT-4 级质量 | ~每百万20美元 | ~每百万0.40美元 | 便宜50倍 |
| 前沿（GPT-5、Claude 4） | — | ~每百万3-10美元 | 新层级 |

大部分改进来自服务效率——第17阶段·04-09的核心课程转化为提供商侧的成本下降。路由让你在应用层捕获这些收益，而不是等待所有用户迁移到廉价层级。

### 漂移是真正的风险

你的路由将40%发送到廉价模型。六个月后，任务分布发生了变化（用户变得更复杂，问更长的的问题）。路由器没有注意到，因为它的分类器是在第一季度的数据上训练的。质量在无声中下降。没有人抱怨得够大声。你在竞争对手的基准测试中发现时已经输了。

通过在线质量指标把关路由：

- 每条路由的用户点赞/点踩。
- 在每条路由的保留样本（5%）上使用自动化 LLM-judge 评估。
- 升级率：如果级联向上路由 >30%，说明廉价模型被过度路由。
- 每条路由的拒绝率。

### 你应该记住的数字

- 2026年同等质量下路由节省：案例研究中20-60%。
- LLM 价格下降2022-2026：总体每年约10倍。
- GPT-4 级别2022 vs 2026：约每百万20美元 → 约每百万0.40美元。
- 级联延迟影响：中位数约1.2倍，升级时约2倍（约10%流量）。

## 关键术语

| 术语 | 人们说的 | 实际含义 |
|------|----------|----------|
| 模型路由 | "成本代理" | 每个请求动态选择模型 |
| 模型级联 | "廉价优先升级" | 先用廉价模型，低置信度时升级到前沿 |
| 预路由 | "先分类" | 前置分类器；无需重试 |
| 集成路由 | "并行选择" | 运行多个，奖励模型选择最佳 |
| 升级率 | "向上路由%" | 级联中升级的请求比例 |
| RouteLLM | "LMSYS 路由器" | 开源路由库 |
| Not Diamond | "商业路由器" | SaaS 模型路由产品 |
| 漂移 | "廉价蔓延" | 分布变化但路由器未察觉 |
| 在线质量门 | "实时检查" | 自动化 LLM-judge 采样实时流量 |
