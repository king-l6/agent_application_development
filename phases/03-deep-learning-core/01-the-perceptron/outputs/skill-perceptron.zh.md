# 感知机模式

感知机计算输入的加权和加偏置，然后应用阶跃函数产生二值输出。它是神经网络的基本单元。

```
output = step(w1*x1 + w2*x2 + ... + wn*xn + bias)
```

## 何时单个感知机足够

- 问题是线性可分的：一条直线（或超平面）可以划分两个类别
- 逻辑门：AND、OR、NOT、NAND
- 简单的阈值决策："分数是否超过X？"
- 在汇集到两个不重叠区域的数据上的二值分类器

## 何时需要多层

- 问题不是线性可分的：没有一条线可以分离类别
- XOR和奇偶问题
- 任何需要"这个但不是那个"推理的任务（条件的组合）
- 现实世界的分类：图像、文本、音频——几乎总是非线性的

## 决策清单

1. 绘制或检查你的数据。你能在类别之间画一条直线边界吗？
   - 能：单层感知机有效
   - 不能：你至少需要两层
2. 这个问题是否可以分解为更简单线性决策的AND/OR组合？
   - 这种分解告诉你最小的网络结构
   - XOR = (A OR B) AND (NOT (A AND B)) = 2层中的3个感知机
3. 对于超过两个类别的问题，每个类别需要一个输出节点

## 训练规则

```
error = expected - predicted
weight_new = weight_old + learning_rate * error * input
bias_new = bias_old + learning_rate * error
```

如果预测正确，什么都不变。如果错误，权重移动以减少误差。这只对单层感知机有效。多层网络需要反向传播。

## 常见错误

- 试图用单个感知机学习非线性模式（它永远不会收敛）
- 设置学习率太高（权重振荡）或太低（训练需要永久）
- 忘记偏置项（没有它，决策边界必须通过原点）
- 混淆感知机收敛（线性可分数据保证收敛）与一般神经网络收敛（不保证）
