# 凸性检查器

如何验证优化问题是否为凸问题，以及如何处理结果。

## 决策检查清单

1. 目标函数是否是凸的？（检查海森矩阵半正定性或使用组合规则。）
2. 所有不等式约束是否为g_i(x) <= 0的形式，且每个g_i是凸的？
3. 所有等式约束是否为仿射（线性）的？
4. 如果以上三个都是肯定的，问题就是凸的。使用具有收敛保证的凸求解器。
5. 如果任何一个是否定的，问题就是非凸的。使用SGD/Adam并接受局部最优。

## 如何检验函数的凸性

| 检验方法 | 适用范围 | 方法 |
|---|---|---|
| 二阶导数 >= 0 | 标量函数 f(x) | 计算f''(x)。如果对所有x有f''(x) >= 0，则为凸函数。 |
| 海森矩阵半正定 | 多变量函数 f(x) | 计算H(x)。如果所有特征值处处 >= 0，则为凸函数。 |
| 定义检验 | 任何函数 | 对采样的x、y、t检查f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y)。 |
| 组合规则 | 组合函数 | 参见下面的组合表。 |
| 直线限制 | 多变量f | f是凸的当且仅当对所有x、v，g(t) = f(x + tv)关于t是凸的。 |

## 组合规则（保持凸性）

| 操作 | 结果 |
|---|---|
| f + g（两者都凸） | 凸 |
| c * f（c > 0，f凸） | 凸 |
| max(f, g)（两者都凸） | 凸 |
| f(Ax + b)，其中f是凸的 | 凸 |
| g(f(x))，其中g是凸非减函数且f是凸的 | 凸 |
| g(f(x))，其中g是凸非增函数且f是凹的 | 凸 |
| 凸函数之和 | 凸 |
| 凸函数的逐点上确界 | 凸 |

## 常见ML目标：凸还是不凸？

| 目标 | 凸？ | 原因 |
|---|---|---|
| MSE: (1/n) sum(y - Xw)^2 | 是 | 关于w是二次的，海森矩阵 = (2/n) X^T X 半正定 |
| 逻辑损失: sum(log(1 + exp(-y_i * w^T x_i))) | 是 | 凸函数之和（log-sum-exp族） |
| 合页损失: sum(max(0, 1 - y_i * w^T x_i)) | 是 | 凸（线性）函数的最大值 |
| L2正则化: lambda * ||w||^2 | 是 | 二次的，海森矩阵 = 2*lambda*I |
| L1正则化: lambda * ||w||_1 | 是 | 绝对值之和（凸但不可微） |
| 岭回归: MSE + L2 | 是 | 两个凸函数之和 |
| LASSO: MSE + L1 | 是 | 两个凸函数之和 |
| Elastic net: MSE + L1 + L2 | 是 | 凸函数之和 |
| SVM（原始形式）: 合页 + L2 | 是 | 凸函数之和 |
| 带softmax的交叉熵 | 是（在logits上） | Log-sum-exp是凸的 |
| 神经网络（任何损失） | 否 | 非线性激活产生非凸组合 |
| k-means目标 | 否 | 离散分配步骤 |
| 矩阵分解: ||X - UV^T||^2 | 否 | 关于U和V是双线性的 |
| GAN损失 | 否 | 极小极大，生成器中非凸 |
| 对比损失（InfoNCE） | 否 | 带负样本的指数比率对数 |

## 基于凸性的求解器选择

| 问题类型 | 求解器 | 收敛保证 |
|---|---|---|
| 凸、光滑、无约束 | 梯度下降 | O(1/k)到全局最小值 |
| 凸、光滑、无约束 | L-BFGS | 超线性到全局最小值 |
| 凸、光滑、无约束 | 牛顿法 | 最小值附近二次收敛（如果海森矩阵可处理） |
| 凸、光滑、带约束 | 内点法 | 多项式时间 |
| 凸、非光滑（L1） | 近端梯度 / ISTA | O(1/k)到全局最小值 |
| 凸、非光滑（L1） | ADMM | 灵活，处理约束 |
| 凸、二次 | 共轭梯度 | n步内精确 |
| 非凸、光滑 | SGD / Adam | 收敛到局部最小值 |
| 非凸、光滑 | SGD + 重启 | 平均得到更好的局部最小值 |
| 非凸、光滑 | 过参数化 + SGD | 平坦最小值，良好的泛化 |

## 常见错误

- 因为损失函数是凸的就认为问题是凸的。损失必须在你正在优化的参数上是凸的。交叉熵在logits上是凸的，但从输入到logits的完整神经网络映射是非凸的。
- 在非凸问题上使用牛顿法。海森矩阵可能有负特征值，导致牛顿法向鞍点或最大值而不是最小值移动。
- 忘记L1正则化使目标函数在零点不可微。标准梯度下降效果不佳。使用近端梯度下降或次梯度方法。
- 通过形成A^T A来平方条件数。如果你需要解决最小二乘问题且A是病态的，使用QR或SVD而不是正规方程。
- 未加检验就声明问题为非凸。许多ML问题（线性模型、SVM、逻辑回归）是凸的，受益于更强的求解器。

## 快速测试：我的问题是凸的吗？

```
1. 写出目标：在约束下最小化f(w)
2. 对于f(w)中的每一项：
   - 它是具有半正定矩阵的二次项吗？ -> 凸
   - 它是一个范数吗？ -> 凸
   - 它是log-sum-exp吗？ -> 凸
   - 它涉及w的非线性（sigmoid(w)，w1*w2）？ -> 可能非凸
3. 所有约束是线性的或凸不等式吗？
4. 如果所有项都是凸的且约束是凸的/线性的 -> 问题是凸的
5. 如果任何项是非凸的 -> 问题是非凸的
```
