# 概率分布选择

如何在建模数据、设计损失函数或设置先验时选择正确的分布。

## 决策清单

1. 结果是离散的（类别、计数）还是连续的（测量值、分数）？
2. 结果是有界的（例如 [0, 1]）还是无界的？
3. 有多少个可能的结果？两个？k 个？无穷？
4. 数据是对称的还是偏斜的？
5. 事件是独立的还是相关的？
6. 你在建模速率、计数、比例还是测量值？

## 分布决策树

```
变量是离散的吗？
  是 --> 只有 2 个结果？ --> 伯努利分布 (p)
     |    k 个结果，一次试验？ --> 分类分布 (p1...pk)
     |    k 个结果，n 次试验？ --> 多项分布 (n, p1...pk)
     |    n 次试验中成功的次数？ --> 二项分布 (n, p)
     |   每个区间内事件的计数？ --> 泊松分布 (lambda)
     |   直到首次成功的试验次数？ --> 几何分布 (p)
     |   直到 r 次成功的试验次数？ --> 负二项分布 (r, p)
  否 --> 对称的钟形？ --> 正态分布 (mu, sigma)
     |   正值，右偏？ --> 对数正态或指数分布
     |   有界于 [0, 1]？ --> Beta 分布 (alpha, beta)
     |   正值，灵活形状？ --> Gamma 分布 (alpha, beta)
     |   事件间的时间？ --> 指数分布 (lambda)
     |   需要重尾？ --> Student's t (nu) 或 Cauchy 分布
     |   多变量，钟形？ --> 多元正态分布
     |   在单纯形上（和为 1）？ --> Dirichlet 分布 (alpha)
```

## 将现实世界的 ML 场景映射到分布

| 场景 | 分布 | 参数 |
|---|---|---|
| 二分类输出 | 伯努利 | p = sigmoid(logit) |
| 多分类输出 | 分类 | p = softmax(logits) |
| 语言模型中的 token 预测 | 词汇表上的分类 | 来自 softmax 的 p |
| 像素强度（归一化） | Beta 或均匀 [0, 1] | 取决于图像统计 |
| 文档中的词数 | 泊松 | lambda = 平均词数 |
| 用户请求间的时间 | 指数 | lambda = 请求率 |
| 测量误差 | 正态 | mu = 0，sigma 来自数据 |
| 权重初始化 | 正态或均匀 | Kaiming/Xavier 规则 |
| VAE 潜空间先验 | 标准正态 | mu = 0，sigma = 1 |
| 比例的贝叶斯先验 | Beta | alpha, beta 来自信念 |
| 类别权重的贝叶斯先验 | Dirichlet | alpha 向量 |
| 回归目标中的噪声 | 正态 | mu = 0，sigma 估计 |
| 对异常值稳健的回归 | Student's t | 自由度低 |
| 持续时间/寿命建模 | Weibull 或 Gamma | 形状和尺度 |
| 每个文档的主题分布（LDA） | Dirichlet | alpha < 1 用于稀疏 |

## 当分布选择出错时

- 在数据有硬性下限（如价格、距离）时使用正态分布。正态分布对负值赋予非零概率。改用对数正态或 gamma 分布。
- 在方差与均值不同时使用泊松分布。泊松假设均值 = 方差。如果方差 > 均值，使用负二项分布。
- 对多类问题使用伯努利分布。伯努利严格是二值的。对于 k > 2 使用分类分布。
- 假设观测值独立而它们实际上是相关的。时间序列、空间数据和分组数据违反独立性。使用自回归或层次模型。

## 常见错误

- 将 PDF 值与概率混淆。PDF 可以超过 1。概率来自于在一个区间上对 PDF 积分。
- 忘记 softmax 输出是分类概率，而不是独立的伯努利概率。它们总和为 1 是构造决定的。
- 在有领域知识时使用均匀先验。如果选择得当，信息先验可以在不偏斜结果的情况下减少方差。
- 将对数概率当作概率处理。对数概率总是负的（或零）。它们之和不等于 1。

## 快速参考：分布性质

| 分布 | 支撑集 | 均值 | 方差 | 关键性质 |
|---|---|---|---|---|
| 伯努利(p) | {0, 1} | p | p(1-p) | 最简单的离散分布 |
| 二项(n, p) | {0..n} | np | np(1-p) | n 个伯努利之和 |
| 泊松(lam) | {0, 1, 2, ...} | lam | lam | 均值 = 方差 |
| 正态(mu, s^2) | (-inf, inf) | mu | s^2 | 给定均值/方差下最大熵 |
| 指数(lam) | [0, inf) | 1/lam | 1/lam^2 | 无记忆性 |
| Beta(a, b) | [0, 1] | a/(a+b) | ab/((a+b)^2(a+b+1)) | 二项分布的共轭先验 |
| Gamma(a, b) | (0, inf) | a/b | a/b^2 | 泊松的共轭先验 |
| Dirichlet(alpha) | 单纯形 | alpha_i/sum | (见公式) | 分类分布的共轭先验 |
