# 采样策略选择

如何为文本生成、贝叶斯推理、蒙特卡洛估计和训练选择合适的采样方法。

## 决策清单

1. 你是要生成输出（文本、图像）还是估计一个量（积分、期望值）？
2. 你能直接从目标分布中采样，还是只能评估其密度？
3. 目标分布是离散的还是连续的？
4. 样本空间的维度是多少？低（< 5）、中（5-100）还是高（> 100）？
5. 你需要精确样本还是近似样本？
6. 你需要通过采样操作获得梯度吗？

## 何时使用每种方法

| 方法 | 何时使用 | 复杂度 | 精确？ |
|---|---|---|---|
| 直接采样 | 你有 CDF 或可以使用库函数 | O(1) 每样本 | 是 |
| 逆 CDF | 已知闭式 CDF 逆（指数、柯西） | O(1) 每样本 | 是 |
| Box-Muller | 需要正态样本但没有库 | O(1) 每样本 | 是 |
| 拒绝采样 | 可以计算目标 PDF，低维度（1-3） | O(1/接受率) 每样本 | 是 |
| 重要性采样 | 需要期望值，而不是单个样本 | O(n) 对于 n 个样本 | 近似 |
| 分层采样 | 蒙特卡洛估计，想要更低方差 | O(n) 对于 n 个样本 | 近似 |
| Metropolis-Hastings | 高维，可以计算未归一化密度 | O(1) 每步 + 燃烧期 | 渐近 |
| Gibbs 采样 | 可以从每个条件分布中采样 | O(d) 每完整扫描 | 渐近 |
| HMC/NUTS | 高维连续，光滑密度 | O(L * d) 每步 | 渐近 |
| 温度采样 | LLM 文本生成，控制创造力 | O(V) 对于词汇表大小 V | 不适用 |
| Top-k 采样 | LLM 生成，移除不可能词元 | O(V log k) | 不适用 |
| Top-p（核）采样 | LLM 生成，自适应候选集 | O(V log V) | 不适用 |
| 重参数化技巧 | 需要高斯采样的梯度（VAE） | O(d) | 是 |
| Gumbel-Softmax | 需要分类采样的梯度 | O(k) 对于 k 个类别 | 近似 |

## LLM 生成设置

| 用例 | 温度 | Top-p | Top-k | 备注 |
|---|---|---|---|---|
| 事实型问答 | 0.0（贪心） | -- | -- | 确定性的，无随机性 |
| 代码生成 | 0.2-0.5 | 0.9 | -- | 低创造力，高连贯性 |
| 一般聊天 | 0.7 | 0.9 | -- | 均衡 |
| 创意写作 | 0.9-1.2 | 0.95 | -- | 较高多样性 |
| 头脑风暴 | 1.0-1.5 | 0.95 | -- | 最大多样性，可能失去连贯性 |

温度和 top-p 可以结合使用。先应用温度（缩放 logits），然后应用 top-p 过滤。

## MCMC 方法选择

| 性质 | Metropolis-Hastings | Gibbs | HMC/NUTS |
|---|---|---|---|
| 维度 | 任意 | 任意（最佳 < 100） | 高（100+） |
| 需要条件分布 | 否 | 是 | 否 |
| 需要梯度 | 否 | 否 | 是 |
| 接受率 | 调整至 ~23% | 总是 100% | 调整至 ~65% |
| 相关性 | 高（随机游走） | 中等 | 低 |
| 燃烧期 | 长 | 中等 | 短 |
| 最适合 | 探索、简单模型 | 共轭模型、贝叶斯网络 | 连续后验、深度概率模型 |

## 常见错误

- 在高维中使用拒绝采样。接受率随维度呈指数下降。超过 5 维，切换到 MCMC。
- MCMC 提议方差设置过高或过低。过高：大多数提议被拒绝，链卡住。过低：所有提议被接受，链移动缓慢。随机游走 MH 的目标接受率约为 23%。
- 忘记燃烧期。MCMC 的前 N 个样本受到起始点的偏倚。至少丢弃 1000 步（对于复杂分布需要更多）。
- 使用与目标差异很大的提议进行重要性采样。少数样本获得巨大权重，使估计不可靠。监控有效样本量：ESS = (sum w_i)^2 / sum(w_i^2)。
- 对需要确定性输出的任务（如分类、结构化提取）使用温度 > 0。使用贪心（T=0）或束搜索代替。
- 不将 top-p 与温度结合使用。单独使用温度无法从长尾中移除垃圾词元。Top-p 可以。
- 通过标准采样操作进行反向传播。对连续（高斯）使用重参数化技巧，对离散（分类）使用 Gumbel-Softmax。

## 快速参考：方差缩减技术

| 技术 | 工作原理 | 方差缩减 |
|---|---|---|
| 分层采样 | 将空间划分为层，每层采样 | 总是 <= 标准 MC |
| 对偶变量法 | 同时使用 U 和 1-U | 适用于单调函数 |
| 控制变量法 | 减去一个已知均值的变量 | 与相关性成正比 |
| 重要性采样 | 从更好的提议中对样本重新加权 | 取决于提议质量 |
| 拉丁超立方体 | 独立地对每个维度分层 | 在高维中优于分层采样 |
