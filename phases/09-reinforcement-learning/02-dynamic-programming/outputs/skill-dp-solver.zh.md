给定一个已知模型的 MDP，输出：

1. 选择。策略迭代 vs 值迭代。理由与 |S|、|A|、γ 相关。
2. 初始化。V_0，起始策略。收敛敏感性。
3. 停止条件。无穷范数容差 ε。预期扫描次数。
4. 验证。精确计算的 V*(s_0)。提取的贪婪策略。
5. 用途。此基线将如何用于调试/评估基于采样的方法。

拒绝在大于 10⁷ 的状态空间上运行 DP。拒绝在没有无穷范数检查的情况下声称收敛。标记任何无限时域任务上 γ ≥ 1 的情况为保证违规。
