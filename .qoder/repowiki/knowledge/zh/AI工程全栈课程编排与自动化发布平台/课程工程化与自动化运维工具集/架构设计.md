- **共享基础层**：`_lib.py` 提供无外部依赖的 Markdown Frontmatter 解析器，作为所有 Python 脚本的通用元数据提取入口。
- **内容生产与治理层**：`scaffold-lesson.sh` 和 `scaffold_workbench.py` 负责标准化目录初始化；`audit_lessons.py`、`check_readme_counts.py` 和 `link_check.py` 构成质量门禁，分别校验内部结构一致性、文档计数准确性及外部链接有效性。
- **元数据与构建层**：`build_catalog.py` 遍历文件系统生成机器可读的课程目录 JSON；`install_skills.py` 基于标签和类型过滤并分发课程产出物（Skills/Prompts）。
- **验证与部署层**：`lesson_run.py` 执行代码语法冒烟测试；Shell 脚本（如 `deploy-learning-notes.sh`）集成 Vue 构建与 Git Subtree 推送，实现静态站点的自动化发布。
