- 采用 Vue 3 Composition API (`<script setup>`) 与 TypeScript 进行开发，通过 `vite.config.ts` 配置路径别名 `@` 指向 `src` 目录。
- 架构上遵循“API层-状态层-组件层”分离：`api/guardrails.ts` 封装所有后端 RESTful 请求（含超时控制），`types/index.ts` 定义全量数据契约，`App.vue` 作为根容器管理全局状态（如拦截历史、统计信息）并协调子组件通信。
- 界面布局采用响应式三栏设计（左侧适配器树、中间交互面板、右侧历史记录），并通过 Tab 切换支持普通对话、对比模式、MCP工具、课程实验台（Playground）及数据库检查点查看器等多种业务场景。
- 组件化程度高，`AdapterTree` 负责可视化展示护栏层级与开关控制，`ChatPanel` 处理多模态对话流与检测结果渲染，`PlaygroundPanel` 实现基于 Schema 的动态表单生成与结果块（Block）渲染。
