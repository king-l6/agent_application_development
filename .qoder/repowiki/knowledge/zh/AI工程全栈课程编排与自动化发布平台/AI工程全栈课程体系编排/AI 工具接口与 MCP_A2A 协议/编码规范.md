- 所有工具定义必须包含机器可读的名称、自然语言描述以及符合 JSON Schema Draft 2020-12 的输入参数约束。
- 工具执行结果必须封装为结构化内容块（如 text/image/resource），并明确区分纯读取工具与具有副作用的后果性工具。
- MCP 交互必须遵循三阶段生命周期（initialize 能力协商 -> operation 双向通信 -> shutdown），并在初始化时声明支持的原子能力。
- 远程 MCP 传输应采用 Streamable HTTP 单端点模式，通过 `Mcp-Session-Id` 维持会话状态并实施 Origin 白名单校验以防御 DNS 重绑定攻击。
