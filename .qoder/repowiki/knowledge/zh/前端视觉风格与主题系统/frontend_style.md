该仓库包含两套独立的前端视觉体系，分别服务于**静态课程站点**和**交互式实验沙箱**。

### 1. 静态课程站点 (site/)
采用**复古技术手册（Retro Technical Manual）**风格，强调高对比度、等宽字体与纸质质感。

*   **设计语言**：
    *   **排版**：标题使用 `VT323`（像素风），正文使用 `Source Serif 4`（衬线体），代码使用 `JetBrains Mono`。这种组合营造出“打印文档”与“终端输出”混合的学术感。
    *   **色彩**：以米白色 (`#fafaf5`) 为底，搭配深灰墨水色 (`#1a1a1a`) 和蓝图蓝 (`#3553ff`) 作为强调色。背景带有微弱的点阵纹理 (`radial-gradient`) 模拟纸张。
    *   **组件**：按钮和卡片采用硬边直角或极小圆角，边框为实线黑框，悬停时产生颜色反转或下划线效果，无阴影或仅使用硬阴影 (`3px 3px 0 var(--ink)`).
*   **主题系统**：
    *   通过 CSS 变量 (`:root` 和 `[data-theme="dark"]`) 实现亮/暗模式切换。
    *   暗色模式转为深蓝底色 (`#0a0d1a`) 配合高亮文字，保持高对比度。
*   **响应式**：使用 `clamp()` 函数实现流体排版，针对移动端隐藏部分导航并调整网格布局。

### 2. 交互式实验沙箱 (guardrails-sandbox/ & site/vue-app/)
采用**现代深色仪表盘（Modern Dark Dashboard）**风格，专注于数据可视化和操作效率。

*   **设计语言**：
    *   **色彩**：基于 Slate 色系（`#0f172a` 背景, `#1e293b` 表面），搭配天蓝色 (`#38bdf8`) 作为主交互色，绿色/红色用于状态指示。
    *   **质感**：使用中等圆角 (`8px-16px`)，细微的边框分隔，以及柔和的过渡动画。
    *   **布局**：典型的三栏式布局（左侧导航/树状图，中间主操作区，右侧历史/详情），适应高密度信息展示。
*   **技术实现**：
    *   基于 Vue 3 + Vite。
    *   样式主要采用 `<style scoped>` 局部作用域 CSS，配合组件内定义的 CSS 变量（如 `--pg-bg`, `--pg-accent`）来隔离主题，避免受全局样式影响。

### 开发规范
1.  **无 CSS 框架**：全站未使用 Tailwind CSS、Bootstrap 或 UI 组件库，所有样式均为手写原生 CSS。
2.  **变量驱动**：所有颜色、字体、间距必须通过 CSS 变量定义，严禁硬编码颜色值，以支持主题切换。
3.  **字体加载**：静态站点依赖 Google Fonts 预连接，确保 `VT323` 和 `Source Serif 4` 的正确渲染。
4.  **隔离原则**：Vue 组件内的实验台样式应使用局部变量前缀（如 `--pg-`），防止与笔记页面的浅色主题冲突。
